# Geometry Library (Test Task)

Простая библиотека на **TypeScript**, реализующая базовые геометрические фигуры и фабрику для их создания.

---

## Описание

Библиотека позволяет создавать различные геометрические фигуры — **круг**, **прямоугольник** и **треугольник** —
и вычислять их **площадь** и **периметр**.
Реализована фабрика `ShapeFactory`, создающая объекты фигур по типу и параметрам.

---

## Реализовано

- [x] Интерфейс `IShape` с базовыми методами `getArea()` и `getPerimeter()`
- [x] Абстрактный класс `Shape` (в `src/abstracts/Shape.ts`)
- [x] Реализация фигур:
  - `Circle` — по радиусу
  - `Rectangle` — по ширине и высоте
  - `Triangle` — по трём сторонам
- [x] Фабрика `ShapeFactory` для централизованного создания фигур
- [x] Создание фигуры из JSON (`ShapeFactory.fromJSON`)
- [x] Покрытие тестами с помощью **Vitest**
- [x] Реализация метода `emitChange()` для уведомления подписчиков об изменениях в фигурах
